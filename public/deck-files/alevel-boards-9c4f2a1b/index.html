CAIE · Cambridge

CAIE 剑桥评估体系线性连贯 · 加权总分 + PUM 相对位置 · 13 个月结转铁律

1评估路径:线性 vs 阶段Linear Route / Staged Route+

Path 1 · 线性路径

Year 12 June→AS + A2 全部组件同时考

认知负荷极大,适合顶尖学生。

→ 合成 A-Level。压力分散。

3合分算法:加权系数 + PUMWeighted marks → PUM percentile+
原始分 49×系数1.4→加权 68.6向上取整→最终 69累加→大纲总分 240/280
加权系数实例:ICT 0417 ×1.4Eco 0455 ×1.5Geo 0460 ×1.333系数每考季微调

⚠ 致命误区:PUM ≠ 答对百分比

PUM = 在该等级段内的相对位置。A* 90–100 / A 80–89 / B 70–79 / C 60–69。PUM 85（A 级）= 处于 A 与 A* 分数线正中间,并非答对了 85% 的题目。逆向工程:PUM + 官方分数线 → 推算原始总分 → 评估复议价值。

核心洞察:CAIE 看重整体加权总分,单一组件等级无法直接推算最终大纲等级。
Edexcel · Pearson

Edexcel 模块化评估体系UMS 标准分 · Cash-in 结算 · "最近两次最优" 陷阱 · 数学聚合算法

1模块化 + UMS 非线性转换6 units · cross-session equating+
  • 独立考试分散压力 · 无限重考逐个击破 · 自由组合战略灵活
  • 4 单元科目 = 400 UMS（物理/化学）· 6 单元 = 600 UMS（生物/数学）

UMS 非线性转换

不同考季难度波动 → 难卷 53 分 = 易卷 58 分 = 100 UMS。相同 UMS = 相同学术水平。最终评级 100% 基于 UMS 总分,原始分仅为中间变量。顾问必知:学生成绩单显示 UMS,不是原始分。

2Cash-in 结算 + 重考陷阱致命:只报单元、漏报结算+
报名代码:W… = 单元考试X… = AS 阶段结算Y… = A-Level 总结算
⚠ 致命失误:只报单元、漏报结算 → 放榜日仅显示零散单元成绩、无 Overall Grade。补救 Late Cash-in（滞纳金 £7.35–50）；影响 UCAS 传输中断、威胁 Unconditional Offer。

⚠ "最近两次最优" 隐蔽陷阱

Chemistry Unit 4:首考 85（优异）→ 二考 70（失常）→ 三考 75。系统只在最近两次取最高 → 锁定 70 vs 75 → 提取 75,早期高分 85 被永久屏蔽。顾问策略:第三次刷分前必须模拟测验确认胜率。所有考试须挂靠同一 UCI。

3数学聚合算法:锁定 vs 解锁Rule 3 Locking / Rule 6 Unlocking+

数学与进阶数学共享单元库（P1–4 必修 / FP1–3 进阶 / M·S·D 应用）。Rule 1:先最大化等级,再最大化 UMS;Rule 2:科目优先级。

🔒 Rule 3 锁定灾难

仅报单一 Cash-in → 满分 M2 被锁进基础数学 → 进阶数学只能用较差单元 → 死锁,机会浪费。

🔓 Rule 6 解锁正解

同时报名 YMA01 + YFM01（双重 Cash-in）→ 解锁全部历史单元 → 全局动态最优组合 → 双 B 或 一 A 一 B。

顾问价值:掌握 Unlocking 是避免高分低报的核心,数学双修必须同时触发双科 Cash-in。
Edexcel 核心风险:行政失误（漏报 Cash-in）与盲目刷分（第三次陷阱）是两大雷区。
OxfordAQA

OxfordAQA 平衡型架构同源 Edexcel 的 UMS · 非对称权重 · 备考生态结构性匮乏

1非对称权重 + 关键特征AS 40% / A2 60%（头轻脚重）+

Psychology / English（4-unit）:Unit1 20% · Unit2 20% · Unit3 30% · Unit4 30% → AS 40% / A2 60%。对比 Edexcel 均等权重,OxfordAQA 强化高年级进阶能力。

  • 同源 Edexcel:UMS 机制完全一致 · A* 标准:总分 80% + A2 单元 90%
  • 模块重考无限次自由组合 · 语言友好:题干素材对国际生友好
2生态限制与战略风险（冰山）表面优势 vs 隐藏风险+

🧊 表面优势

  • 试题较直白
  • 分数线可能友好
  • 部分科目出分率优异

⚠ 隐藏风险

  • 真题资源极度稀缺（< CAIE/Edexcel 的 30%）
  • 分类题库几乎不存在;核心资源锁在 Centre Extranet
  • 社会考生接受度受限（仅 British Council 渠道,不含实验科目）
⚠ 顾问警示:不可仅因"听说 AQA 简单"而盲目转局,必须评估资源支持。脱离成熟学校体系的学生 = 巧妇难为无米之炊。
OxfordAQA 悖论:试题本身可能友好,但备考生态的结构性匮乏使其成为"资源依赖型"选择。
Hard Boundary

跨考试局组合禁忌刚性壁垒 · A* 评定三局对比 · UCAS 申报验证

🚫 跨局结转绝对禁止

CAIE ⇄ Edexcel ⇄ OxfordAQA 互相不可结转、禁止成绩拼凑。底层不兼容:计算逻辑根本差异（PUM vs UMS）、大纲知识点分布不同、评估目标不匹配、学分转移已全面废止。

A* 评定三局逻辑

  • CAIE:全局单线——总分 ≥ A* 门槛即可,AS 高分可补偿 A2（高容错）
  • Edexcel:双重刚性——总分 ≥80% 且 A2 ≥90%,缺一不可
  • OxfordAQA:同源 Edexcel 逻辑,更偏重 A2

UCAS 申报验证

Awarding Body 必须精确填写 → 大学自动化系统交叉验证 → Results Day 匹配。考试局信息不匹配 = 验证失败 / 审核延误 / Offer 无法兑现。常见失误:填错考试局名称、改考点未更新 Centre Number、混淆科目代码。补救:Qualification Amendment Form(最长 21 天)。

顾问责任:转学规划必须评估考试局连续性,UCAS 填报必须逐项核对考试局信息,避免验证失败。
