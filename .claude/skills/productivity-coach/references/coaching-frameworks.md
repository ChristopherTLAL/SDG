# 教练框架库（productivity-coach 的 L2 深挖时读）

L0/L1 用不到这么细；当世杰真想深聊、或要做决策台 / 事前验尸 / 季度审时，来这取对应框架。每条 = 精髓 + 怎么当探针用。**锚永远是他的目标函数**（`效率模型.md`）。

## 经济学
- **机会成本 / 时间影子价格**：每小时真实成本 = 它最好的替代用途。估他时薪（营收 ÷ 可计费小时）→ 任何「自己做 vs 外包」都用它定价。探针：「这 2h 套版按你时薪≈¥X，外包 ¥Y < 2X 就外包」。
- **比较优势（Ricardo）**：哪怕你样样比顾问强（绝对优势），也该专精在**相对**最强处、其余交易出去。探针：「你在文书终审比在套版强得更多——套版让出去，哪怕你做得更好」。
- **边际 / 递减回报**：第 5 小时的文书 ≠ 第 1 小时。探针：「PS 已 A−，第 3 遍只到 A 却吃 2h——那 2h 救个转红学生回报更高」。
- **沉没成本 + 损失厌恶（Kahneman）**：过去投入不该绑架未来决策。探针：「这单已投 20h，只看往后的 ROI，该续吗？」
- **委托-代理 / 激励（principal-agent）**：交办要对齐激励 + 留检查点，否则回旋镖。探针：「交解航前给清规格 + 一个 check-in，不然又弹回来」。
- **期望值 / 组合（EV）**：申请是概率的（剑桥≈10%）。边际小时投到组合里 EV 最高处。探针：「50 个学生，这小时投谁产出最高？」
- **Goodhart 定律**：指标变目标就失真（二销 KPI → 推不需要的产品 → churn）。探针：「这步是为学生还是为 KPI？」

## 项目管理
- **关键路径（CPM）**：决定 deadline 的那条依赖链。探针：「李子萱剑桥的瓶颈是文书（等素材表 → 你 3 次面谈），现在不启 10/15 滑档」。
- **关键链 / 缓冲管理（Goldratt）**：别每任务都加 buffer，池在项目级 + 盯消耗。探针：「到雅思出分 18 天缓冲、已烧 12 = 黄」。
- **在制品上限 / Little 定律**：吞吐 = WIP ÷ 周期；同时挂太多反而都慢。探针：「14 个学生同时在文书阶段——砍到 6 个在跑、其余排队，更快完」。
- **约束理论（ToC, Goldratt）**：系统吞吐 = 瓶颈。一切过你 = 事务所吞吐 = 你的小时数。最高杠杆永远是松绑你这个瓶颈。探针：「这事别再卡你身上，怎么系统化 / 交人 / 自动化？」
- **事前验尸（pre-mortem, Klein）**：动手前先假设它已失败、倒推原因。比事后复盘早且便宜。
- **RACI**：每个学生-任务谁担责 / 批 / 咨询 / 知会——减少「我以为你在做」的漏球。

## 生产力 / 个人效能
- **深度工作（Newport）**：深度活（选校判断 / 文书终审 / 家长策略会）要整块、不可打断；一天现实 2-3 个深度块。浅层活（回消息 / 打勾 / 排期）批处理。
- **Maker vs Manager（Graham）**：你既是 maker 又是 manager，两种日程冲突——分「做事日」和「管理日」。
- **Eisenhower 利刃版**：重要 = 杠杆；陷阱是「紧急不重要」挤掉「重要不紧急」（系统化、带顾问、做内容这些 ON-the-business 的事）。保护后者的时间。
- **GTD 周复盘**：capture→clarify→organize→reflect→engage；周复盘是龙骨习惯。

## 人生 · 成长（最高层，锚定目标函数）
- **衡量你的人生（Christensen, Harvard）**：把时间 / 才华投在你临终会用来衡量自己的尺上（家人 / 改变的学生 / 正直），别被「边际成本」一次次诱惑侵蚀原则。季度审用它。
- **显示性 vs 陈述性偏好（Samuelson）**：你嘴上说的 vs 小时实际去向。我有你 telemetry → 照镜子。
- **职业资本（Newport, So Good They Can't Ignore You）**：你的文书 / 选校判断是稀缺 + 有价值的资本，攒够去兑换自主 / 影响力。盯资本积累，不只任务完成。
- **复利**：习惯（写作 / 读学术 / 复盘——你 TickTick 里就有）+ 资产（声誉 / 内容库 / 团队能力 / 这个 vault 知识库）会复利。把日常动作连到长线资产。
- **反脆弱 / via negativa（Taleb）**：进步常是「停做坏的」而非「多做」；别让一切依赖你（OAuth 封禁那次 = 脆弱性咬人）。保留 optionality。

## Cambridge 对辩模式（怎么聊，不是聊什么）
导师制（supervision）= 1:1 对抗式辅导：你抛半成型观点，我**认真站对立面**打、要你 steelman 反方、用苏格拉底追问逼你把逻辑补全——**不是附和**。L2 默认用这个语气，但仍把决定权留给他。
